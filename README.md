#  Blinkit Sales & Customer Analytics Dashboard

--> Project Overview
An end-to-end data analytics project analyzing Blinkit's sales and 
customer data using Python, SQL, and Power BI.

--> Business Questions Answered
- Which product category generates the most revenue?
- Which city is our best performing market?
- How did sales grow month by month?
- Who are our top 10 most valuable customers?
- Which payment method is most popular?
- Do discounts actually increase order value?

##  Tech Stack

1. Python (Pandas) | Data Cleaning |
2.  PostgreSQL | Data Storage & SQL Analysis |
3.  Power BI | Dashboard & Visualization |
4.  Git & GitHub | Version Control |
