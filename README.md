#  Blinkit Sales & Customer Analytics Dashboard
##  Dashboard Pages
- Page 1 → Executive Summary (KPIs)
- Page 2→ Sales Analysis
- Page 3 → Customer Analysis
- Page 4 → Trends Analysis

--> Data Cleaning Steps
1. Load & explore data
2. Handle missing values
3. Remove duplicates
4. Fix data types
5. Detect & handle outliers
6. Standardize text columns
7. Feature engineering
8. Add business columns
9. Save cleaned data

--> Key Insights
- Personal Care & Staples are top revenue categories
- Mumbai & Delhi are best performing cities
- UPI is the most preferred payment method
- Evening slots (6PM-9PM) get maximum orders
